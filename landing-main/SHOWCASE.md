# Витрина проектов – Как добавить свой проект

Участники сообщества «Тимлид не кодит» могут разместить здесь свой проект – стартап, open-source библиотеку, блог, Telegram-канал или любой другой инструмент.

## Как это работает

Каждый проект – отдельный `.md` файл в папке `landing-main/content/showcase/`. Файл попадает на сайт после мержа Pull Request.

## Пошаговая инструкция

1. **Форкните репозиторий** – https://github.com/belyaevsa/teamleads-2025
2. **Скопируйте шаблон** – файл `landing-main/showcase-template.md`
3. **Назовите файл** латиницей через дефис, например `my-awesome-tool.md`
4. **Заполните frontmatter** – все поля описаны в шаблоне
5. **Напишите описание** в теле файла (Markdown)
6. **Отправьте Pull Request** в ветку `master`

## Формат frontmatter

```yaml
---
title: "Название проекта"
description: "Краткое описание в одну строку"
projectUrl: "https://example.com"
author: "Ваше имя"
date: 2026-05-22  # дата добавления
tags:
  - SaaS         # 1–3 тега: SaaS, Open Source, Блог, Инструмент, Курс, Телеграм-канал, Библиотека
social:
  telegram: ""
  github: ""
  linkedin: ""
  twitter: ""
draft: false
---
```

## Правила

- Один файл – один проект (можно добавить несколько проектов разными файлами)
- Язык – русский
- Описание – конкретное и полезное, без рекламных штампов
- Проект должен быть рабочим (не «скоро запустим»)
- Ссылки должны вести на реальные ресурсы
- Контакты – по желанию, но хотя бы одна ссылка приветствуется

## Что будет после мержа

- Проект появится на странице [teamleads.kz/showcase/](https://teamleads.kz/showcase/)
- На главной странице показываются 6 последних проектов
- У каждого проекта будет своя страница с полным описанием и контактами
