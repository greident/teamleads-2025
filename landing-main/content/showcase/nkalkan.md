Кроссплатформенная поддержка для криптопровайдера АО "НУЦ" KalkanCrypt в .NET. Данная библиотека использует C API, в том числе на Windows, что позволяет иметь общее API на всех платформах которые поддерживает KalkanCrypt. Данная библиотека также распространяется через Nuget для удобстав разработчиков.
 
Бинарные файлы KalkanCrypt не в ходят в комплект поставки и распространяются АО "НУЦ" самостоятельно.
